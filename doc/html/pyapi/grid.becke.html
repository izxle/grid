grid.becke module¶

Becke Weights Module.

class BeckeWeights(radii=None, order=3)¶

Bases: object

Becke weights functions holder class.

__init__(radii=None, order=3)¶

Initialize class.

Parameters

  • radii (dict, optional) – Dictionary of atomic number and corresponding atomic radius. If None, Bragg-Slater empirically measured covalent radii are used.

  • order (int, optional) – Order of iteration for switching function.

generate_weights(points, atom_coords, atom_nums, *, select=None, pt_ind=None)¶

Calculate Becke integration weights of points for select atom.

Parameters

  • points (np.ndarray(N, 3)) – Cartesian coordinates of \(N\) grid points.

  • atom_coords (np.ndarray(M, 3)) – Cartesian coordinates of \(M\) atoms in molecule.

  • atom_nums (np.ndarray(M, 3)) – Atomic number of \(M\) atoms in molecule.

  • select (list or integer, optional) – Index of atom index to calculate Becke weights

  • pt_ind (list, optional) – Index of points for splitting sectors

Returns

Becke integration weights of \(N\) grid points.

Return type

np.ndarray(N, )
