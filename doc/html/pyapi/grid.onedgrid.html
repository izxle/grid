GaussChebyshev(npoints)¶

Generate 1D grid on [-1, 1] interval based on Gauss-Chebyshev quadrature.

The fundamental definition of Gauss-Chebyshev quadrature is:

\[\int_{-1}^{1} \frac{f(x)}{\sqrt{1-x^2}} dx \approx \sum_{i=1}^n w_i f(x_i)\]

However, to integrate function \(g(x)\) over [-1, 1], this is re-written as:

\[\int_{-1}^{1}g(x) dx \approx \sum_{i=1}^n w_i \sqrt{1-x_i^2} g(x_i) = \sum_{i=1}^n w_i' g(x_i)\]
Parameters

npoints (int) – Number of grid points.

Returns

A 1D grid instance.

Return type

OneDGrid

GaussChebyshevLobatto(npoints)¶

Generate 1D grid on [-1, 1] interval based on Gauss-Chebyshev-Lobatto.

The definition of Gauss-Chebyshev-Lobato quadrature is:

\[\int_{-1}^{1} \frac{f(x)}{\sqrt{1-x^2}} dx \approx \sum_{i=1}^n w_i f(x_i)\]

However, to integrate function \(g(x)\) over [-1, 1], this is re-written as:

\[\int_{-1}^{1}g(x) dx \approx \sum_{i=1}^n w_i \sqrt{1-x_i^2} f(x_i) = \sum_{i=1}^n w_i' f(x_i)\]

Where

\[x_i = \cos\left( \frac{(i-1)\pi}{n-1} \right)\]

And the weights

\[w_{1} = w_{n} = \frac{\pi}{2(n-1)}\]

And the internal weights

\[w_{i\neq 1,n} = \frac{\pi}{n-1}\]
Parameters

npoints (int) – Number of points in the grid

Returns

A 1D grid instance.

Return type

OneDGrid

GaussChebyshevType2(npoints)¶

Generate 1D grid on [-1, 1] interval based on Gauss-Chebyshev 2nd kind.

The fundamental definition of Gauss-Chebyshev quadrature is:

\[\int_{-1}^{1} \sqrt{1-x^2} f(x) dx \approx \sum_{i=1}^n w_i f(x_i)\]

However, to integrate function \(g(x)\) over [-1, 1], this is re-written as:

\[\int_{-1}^{1}g(x) dx \approx \sum_{i=1}^n \frac{w_i}{\sqrt{1-x_i^2}} f(x_i) = \sum_{i=1}^n w_i' f(x_i)\]

Where

\[x_i = \cos\left( \frac{i}{n+1} \pi \right)\]

and the weights

\[w_i = \frac{\pi}{n+1} \sin^2 \left( \frac{i}{n+1} \pi \right)\]
Parameters

npoints (int) – Number of grid points.

Returns

A 1D grid instance.

Return type

OneDGrid

GaussLaguerre(npoints, alpha=0)¶

Generate 1D grid on [0, inf) interval based on Generalized Gauss-Laguerre quadrature.

The fundamental definition of Generalized Gauss-Laguerre quadrature is:

\[\int_{0}^{\infty} x^\alpha e^{-x} f(x) dx \approx \sum_{i=1}^n w_i f(x_i)\]

However, to integrate function \(g(x)\) over [0, inf), this is re-written as:

\[\int_{0}^{\infty} g(x)dx \approx \sum_{i=1}^n \frac{w_i}{x_i^\alpha e^{-x_i}} g(x_i) = \sum_{i=1}^n w_i' g(x_i)\]
Parameters

  • npoints (int) – Number of grid points.

  • alpha (float, optional) – Value of parameter \(alpha\) which should be larger than -1.

Returns

A 1D grid instance.

Return type

OneDGrid

RectangleRuleSine(npoints)¶

Generate 1D grid on (0:1) interval based on rectangle rule.

The fundamental definition of this quadrature is:

\[\int_{0}^{1} f(x) dx \approx \sum_{i=1}^n w_i f(x_i)\]

The range of integration can be modified by :math: q = 2 x - 1.

\[2 \int_{0}^{1} f(x) dx = \int_{-1}^{1} f(q) dq\]

Where

\[x_i = \frac{2 i - 1}{2 n}\]

And the weights

\[w_i = \frac{2}{n^2 \pi} \sin(n\pi x_i) \sin^2(n\pi /2) + \frac{4}{n \pi}\sum_{m=1}^{n-1} \frac{\sin(m \pi x_i)\sin^2(m\pi /2)} {m}\]
Parameters

npoints (int) – Number of points in the grid.

Returns

A 1D grid instance.

Return type

OneDGrid

TanhSinh(npoints, delta=0.1)¶

Generate 1D grid on [-1,1] interval based on Tanh-Sinh rule.

The fundamental definition is:

\[\int_{-1}^{1} f(x) dx \approx \sum_{i=-\frac{1}{2}(n-1)}^{\frac{1}{2}(n-1)} w_i f(x_i)\]

Where

\[x_i = \tanh\left( \frac{\pi}{2} \sinh(i\delta) \right)\]

And the weights

\[w_i = \frac{\frac{\pi}{2}\delta \cosh(i\delta)} {\cosh^2(\frac{\pi}{2}\sinh(i\delta))}\]
Parameters

  • npoints (int) – Number of points in the grid, this value must be odd.

  • delta (float) – This values is a parameter \(\delta\), is related with the size.

Returns

A 1D grid instance.

Return type

Trapezoidal(npoints)¶

Generate 1D grid on [-1:1] interval based on trapezoidal rule.

The fundamental definition of this rule is:

\[\int_{-1}^{1} f(x) dx \approx \sum_{i=1}^n w_i f(x_i)\]

Where

\[x_i = -1 + 2 \left(\frac{i-1}{n-1}\right)\]

The weights

\[w_{i\neq 1,n} = \frac{2}{n}\]

and

\[w_1 = w_n = \frac{1}{n}\]
