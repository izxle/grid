interpolate(spline, r_points, theta, phi, deriv=0)¶

Interpolate angular points on given r value.

Parameters

  • spline (scipy.CubicSpline) – CubicSpline object for interpolating function value

  • r_points (float or list[float]) – Radial value to interpolate function values

  • theta (np.ndarray(N,)) – Azimuthal angle of angular grid

  • phi (np.ndarray(N,)) – Polar angle of angular grid

  • deriv (int, default to 0) – 0 for interpolation, 1, 2, 3 for 1st, 2nd and 3rd derivatives

Returns

Interpolated function value at spherical grid

Return type

np.ndarry(N,) or np.ndarray(n, N)
